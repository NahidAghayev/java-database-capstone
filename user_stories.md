# Smart Clinic Management System — User Stories

## Admin User Stories

1. As an Admin, I want to log in with my credentials so that I can access the admin dashboard and manage the system securely.

2. As an Admin, I want to add new doctors with their name, specialization, email, password, phone, and available time slots so that patients can browse and book appointments with them.

3. As an Admin, I want to view all registered doctors so that I can monitor the clinic's medical staff.

4. As an Admin, I want to delete a doctor from the system so that I can remove doctors who are no longer practicing at the clinic.

5. As an Admin, I want to filter doctors by name, available time, and specialty so that I can quickly find specific doctors in the system.

6. As an Admin, I want to log out of the dashboard so that unauthorized users cannot access administrative functions.

## Patient User Stories

7. As a Patient, I want to sign up with my name, email, password, phone, and address so that I can create an account and book appointments.

8. As a Patient, I want to log in with my email and password so that I can access my appointment dashboard securely.

9. As a Patient, I want to browse all available doctors so that I can choose a doctor for my appointment.

10. As a Patient, I want to filter doctors by name, specialty, and available time (AM/PM) so that I can find a doctor that matches my needs and schedule.

11. As a Patient, I want to book an appointment with a selected doctor on a chosen date and time so that I can secure a consultation slot.

12. As a Patient, I want to view all my appointments so that I can keep track of my upcoming and past visits.

13. As a Patient, I want to filter my appointments by status (upcoming or past) and search by doctor or patient name so that I can find specific appointments easily.

14. As a Patient, I want to update the date and time of an existing appointment so that I can reschedule if my plans change.

15. As a Patient, I want to log out of my account so that my session remains secure.

## Doctor User Stories

16. As a Doctor, I want to log in with my email and password so that I can access my dashboard and manage my appointments.

17. As a Doctor, I want to view my appointments for a selected date so that I can see which patients I am seeing that day.

18. As a Doctor, I want to search my appointments by patient name so that I can quickly find a specific patient's appointment.

19. As a Doctor, I want to view a patient's medical record so that I can review their appointment history before a consultation.

20. As a Doctor, I want to add a prescription for a patient (including medication names, dosage, and notes) so that the patient receives the necessary treatment plan.

21. As a Doctor, I want to view existing prescriptions for a patient's appointment so that I can review past recommendations.

22. As a Doctor, I want to log out of my dashboard so that my patient data remains confidential and secure.
